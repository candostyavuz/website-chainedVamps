About Chained Vampires

Chained Vampires are ERC-721 NFT's which returns dividends to their owners! Only 9999 vampires available which each of them is generated randomly with Computer Program. Each vampire has unique apperance, attributes, race and rarity.

TOKENOMICS

Reflectionary minting system returns vampire holders a portion from minting fees, all in $AVAX!

This means that 10% of every minting fee is reflected back to existing holders and can be claimed at any time!
